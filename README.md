# Менеджер задач (Telegram-бот)

Учебный проект для курса Python в Алгоритмике.

### Функции
- Добавление задач (/add)
- Список задач (/list)
- Отметка выполнения (/done)
- Удаление (/delete)
- Админ-панель (/admin_panel)

### Запуск
1. Установи зависимости: `pip install -r requirements.txt`
2. Добавь токен в `.env`
3. Запусти: `python bot_mini_app.py`
